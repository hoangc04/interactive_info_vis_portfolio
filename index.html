Sketch narrative

This sketch visualizes a swimmer lapping around a swimming pool, with the lap representative of a one hour time frame with 5 minute breaks per 15 minute interval.

  • Context: This can be used as a countdown timer for a task, like studying, exercising, while also leaving room for 5 minute breaks. This clock style uses a pomodoro method that I decided to use for all of my clocks.
  • Design decisions: I chose to create a the visual shape of a pool to portray the task of a swimmer completing one full swimming lap, relating to one full hour of time consumption to execute any task, which could be related to sports or other tasks. There are tick marks at every 15 minutes to show the swimmer reaching a 15 minute interval, and a countdown timer appears for the 5 minute break only when the interval is reached.
  • Future work: A possible extension couldbe to create further button options for the user to pick what time frame, 30 min, 1 hour, multiple hours, and change the interval labels based on that choice.

Sketch narrative

Context: This sketch visualizes a melting icecube as a countdown timer.

  • Context: This can be used as a countdown timer for a break, studying, cooking, or any task that needs to be measured by a 5 minute countdown. It can be used as a pomodoro method break in studying, utilizing a 5 minute break period between tasks.
  • Design decisions: I decided to use the concept of a melting icecube because it can be easily imagined as something that has an endpoint, as it finishes melting. I chose to use an expanding puddle to show the time elapsed expanding, as well as droplets to represent the seconds passing.
  • Future work: I think an improvement would be to create a starting trigger a button, so that this countdown can be used at any point in time.

Sketch narrative

Context: This sketch visualizes a train traveling along a train track and stopping at a station stop before reaching its final destination.

  • Context: This clock can be used as a studying tool or any task that needs to be broken up into two different parts, with a break in the middle. It utilizes a pomodoro technique, in this instance for a shorter task,with 3 minute task intervals and a 1 minute task for smaller, shorter tasks.
  • Design decisions: I chose to use a train to visualize a starting point and an end point. This way, there is a clear ending to a task. I chose visual pictures to depict the scene so it can be easily identifiable what the clock is trying to portray.
  • Future work: For future enhancements, I would add an option to extend the task intervals and break time durations, as well as a feature to add or remove stops. This makes it more customizable and can be more applicable to unique tasks.
